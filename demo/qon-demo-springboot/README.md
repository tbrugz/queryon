
qon-demo-springboot
=====

Demo using QueryOn with [Spring Boot](https://spring.io/projects/spring-boot)


running
-----

`mvn spring-boot:run`

OR (jar)

```
mvn clean package
java -jar target/qon-demo-springboot-0.7-SNAPSHOT.jar
```
